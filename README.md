# BCE Python SDK

This repository contains the Python SDK for Baidu Cloud (Baidu Cloud Engine / BCE). The package on PyPI is [`bce-python-sdk`](https://pypi.org/project/bce-python-sdk/).

## Links

- [Baidu Cloud Developer SDK page](https://cloud.baidu.com/doc/Developer/index.html?sdk=python)
- [GitHub repository](https://github.com/baidubce/bce-sdk-python)
- [PyPI package: bce-python-sdk](https://pypi.org/project/bce-python-sdk/)

## Install

```bash
pip install bce-python-sdk
```

## Next generation SDK

The next generation Python SDK is `baiducloud-python-sdk`:

- [GitHub repository: baiducloud-python-sdk](https://github.com/baidubce/baiducloud-python-sdk)
- [PyPI publisher: bce-sdk](https://pypi.org/user/bce-sdk/)

It is published as separate PyPI packages matching the packages in the GitHub repository. For example:

- [baiducloud-python-sdk-core](https://pypi.org/project/baiducloud-python-sdk-core/)
- [baiducloud-python-sdk-vpc](https://pypi.org/project/baiducloud-python-sdk-vpc/)
- [baiducloud-python-sdk-eip](https://pypi.org/project/baiducloud-python-sdk-eip/)

See the GitHub repository and PyPI publisher page for the latest complete package list.
